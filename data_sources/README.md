# Data Sources

This folder stores source and archival datasets that are kept in the repository for reproducibility but are not needed at runtime.

The live site reads only these data files from `public/data/`:

- `geographic_model_data_with_biomes.csv`
- `ultra_optimized/grid_data_minimal.json.gz`
- `ultra_optimized/gmm_metrics_minimal.json.gz`
- `ultra_optimized/traits_data_minimal.json.gz`
- `ultra_optimized/species_grid_data.json.gz`
- `ultra_optimized/species_umap2.json.gz`
